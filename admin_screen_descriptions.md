# 3.1.1.2.1 Admin System - Admin Web Screen

| # | Feature | Screen | Description |
| :--- | :--- | :--- | :--- |
| 1 | Authentication | Login Screen | This screen allows users to log in to the system and access features based on their assigned roles.<br/><br/>**Displayed Information:**<br/>• Input fields for credentials (Email and Password).<br/>• A “Quên mật khẩu” option to initiate the password recovery flow.<br/>• A “QUAY LẠI TRANG CHỦ” option to return to the landing home page.<br/>• A “Đăng nhập” button to submit credentials.<br/>• Password visibility toggle icon.<br/><br/>**Available Actions:**<br/>• Enter email and password credentials.<br/>• Toggle password visibility.<br/>• Click “Đăng nhập” to authenticate and gain system access.<br/>• Click “Quên mật khẩu” to navigate to the password recovery screen.<br/>• Click “QUAY LẠI TRANG CHỦ” to return to the landing home page. |
| 2 | Giám sát Tổng quan Hệ thống (System Overview Monitoring) | Bảng Điều Khiển (Admin Dashboard Screen) | This screen provides a high-level overview of the platform's statistics and therapist activities.<br/><br/>**Displayed Information:**<br/>• Stats cards displaying Total Users (with weekly growth), Total Teachers (with pending account count), and Total VR Practice Attempts (with daily attempts count).<br/>• Recent activities feed showing 5 latest VR attempts (child name, activity type, target lesson/exercise ID, and timestamp).<br/>• Quick-action buttons to navigate directly to other management modules.<br/>• Global search bar and notification indicators.<br/><br/>**Available Actions:**<br/>• Navigate to other management pages using quick-action shortcuts.<br/>• Search the system using the global search input.<br/>• View system notifications. |
| 3 | User Account Management | User Management Screen | This screen allows the admin to view and manage all accounts in the system.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Accounts, Active accounts, Locked accounts, and New Users this month.<br/>• A table displaying the list of users with information such as ID (USR-xxx), Full Name, Contact info (Email & Phone), Role badge, Status (Active/Locked), and Join Date.<br/>• A filter bar to filter accounts by role and status, and a search box.<br/><br/>**Available Actions:**<br/>• Filter accounts using the filter bar.<br/>• Search accounts by name, email, phone number, or username.<br/>• Create a new teacher or parent account (a 2-step flow: choosing role then filling the profile details).<br/>• Edit account details (Full Name, Phone, Gender, Status, and Specialty).<br/>• Lock or unlock accounts.<br/>• Reset account passwords.<br/>• View user details. |
| 4 | Role Management | Role Management Screen | This screen allows the admin to view and manage user roles and capabilities in the system.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Roles, Active Roles, and Inactive Roles.<br/>• A grid layout of role cards displaying Role ID, Role Name (Admin, Teacher, Parent, Child), Description, Status badge (Active/Inactive), and Created Date.<br/>• A filter bar to filter roles by active status.<br/><br/>**Available Actions:**<br/>• Filter roles by status and search by keyword.<br/>• Sort roles list by ID or Role Name (A-Z/Z-A).<br/>• Create a new role.<br/>• Update a role's description and active status.<br/>• Delete a role. |
| 5 | Teacher Management | Teacher Management Screen | This screen allows the admin to monitor and manage special education teacher profiles in the system.<br/><br/>**Displayed Information:**<br/>• A table displaying the list of teachers with information such as Teacher Code (USR-xxx), Full Name, Contact info (Email & Phone), Specialty, Status badge, and Join Date.<br/>• A filter bar to filter teachers by status, and a search box.<br/><br/>**Available Actions:**<br/>• Filter teachers using the filter bar and search by keyword.<br/>• View teacher profile details.<br/>• Edit teacher details (Specialty, Phone, Gender, Status).<br/>• Lock or unlock teacher accounts.<br/>• Navigate to the user creation page to add new teachers. |
| 6 | Parent Management | Parent Management Screen | This screen allows the admin to monitor and manage parent profiles in the system.<br/><br/>**Displayed Information:**<br/>• A table displaying the list of parents with information such as Parent Code (USR-xxx), Full Name, Contact info (Email & Phone), Additional notes, Status badge, and Join Date.<br/>• A filter bar to filter parents by status.<br/><br/>**Available Actions:**<br/>• Filter parents using the filter bar and search by keyword.<br/>• View parent details.<br/>• Edit parent details.<br/>• Lock or unlock parent accounts.<br/>• Navigate to the user creation page to add new parents. |
| 7 | Child Profile Management | Child Profile Management Screen | This screen allows the admin to manage profiles of children using the VR client application.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Children, Active children, and children needing Special Support.<br/>• A table displaying child profiles with Avatar, Child ID (CHD-xxx), Full Name, Age, Gender, Linked Parent Name, Learning Level (Beginner/Intermediate/Advanced), and Status (Active/Inactive).<br/>• A filter bar to filter children by age group, gender, learning level, and status, and a search box.<br/><br/>**Available Actions:**<br/>• Filter children using the filter bar.<br/>• Search children by child name or parent name.<br/>• View child profile details.<br/>• Create a new child profile (specifying linked parent, age, gender, level, avatar, and notes).<br/>• Update child profile details.<br/>• Delete child profiles. |
| 8 | Speech Diagnosis & Therapy Goals | Speech Diagnosis Screen | This screen allows the admin or therapist to manage speech evaluation reports and intervention goals for children.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Assessments, and case counts for Mild, Moderate, and Severe speech delay levels.<br/>• A table displaying assessments with Child Name, Speech Delay Level (Mild/Moderate/Severe), Clinical Diagnosis, Pronunciation Ability (Khả năng phát âm), Assessment Date, and Next Assessment Date.<br/>• A filter bar to filter assessments by speech level, communication ability, and date range.<br/><br/>**Available Actions:**<br/>• Filter assessments using the filter bar and search by child name.<br/>• View detailed assessment reports.<br/>• Create a new speech assessment (evaluating 6 core indicators: communication, pronunciation, comprehension, expression, attention, and social interaction from Very Poor to Excellent; setting diagnosis, strengths, weaknesses, recommendations, and goals).<br/>• Update assessments.<br/>• Delete assessments. |
| 9 | Classroom Management | Classroom Management Screen | This screen allows the admin to create and manage virtual classroom sessions.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Classrooms, Active classrooms, Inactive classrooms, and Total Enrolled Students.<br/>• A table displaying classrooms with Class ID (CLS-xxx), Class Name, Teacher Name, Schedule (Start Date & End Date), Enrolled Student Count, and Status (Active/Inactive/Upcoming/Closed).<br/>• A filter bar to filter classrooms by status, program, and teacher, and a search box.<br/><br/>**Available Actions:**<br/>• Filter classrooms using the filter bar.<br/>• Search classrooms by name.<br/>• Create a new classroom.<br/>• Edit classroom details.<br/>• View classroom details (including program details, teacher info, and a list of enrolled children). |
| 10 | Student Enrollment | Student Enrollment Screen | This screen allows the admin to enroll children into specific classrooms and manage their status.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Enrollments, Active enrollments, Pending approvals, and Completed/Cancelled.<br/>• A table displaying enrollment records with Enrollment ID (ENR-xxx), Child Name, Class Name (Lớp tiếp nhận), Pronunciation Level (Cấp độ phát âm), Enrollment Date, and Status (Active/Completed/Cancelled/Pending).<br/>• A filter bar to filter enrollments by status and classroom.<br/><br/>**Available Actions:**<br/>• Filter enrollments using the filter bar and search by child or class name.<br/>• View enrollment details.<br/>• Enroll a child into a classroom.<br/>• Update enrollment details (changing class, changing status).<br/>• Delete enrollment records. |
| 11 | Semester Management | Semester Management Screen | This screen allows the admin to view and manage semesters for scheduling classrooms.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Semesters, Active Semesters, and Upcoming Semesters.<br/>• A table displaying semesters with Semester ID (SEM-xxx), Semester Name, School Year Name, Responsible Teacher, Class Count, Start Date, End Date, and Status (Active/Upcoming/Completed).<br/>• A filter bar to filter semesters by status and school year.<br/><br/>**Available Actions:**<br/>• Filter semesters using the filter bar.<br/>• Search semesters by name.<br/>• Create a new semester.<br/>• Update a semester.<br/>• Delete a semester. |
| 12 | School Year Management | School Year Management Screen | This screen allows the admin to view and manage school years as timebases for the system.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total School Years, Active school years, and Completed school years.<br/>• A table displaying school years with Year ID (SCH-xxxx), Year Name, Start Date, End Date, Status (Active/Upcoming/Completed), and Created Date.<br/>• A filter bar to filter school years by status.<br/><br/>**Available Actions:**<br/>• Filter school years using the filter bar.<br/>• Search school years by name.<br/>• Create a new school year.<br/>• Update a school year.<br/>• Set active school year (setting it as the main active year for the system).<br/>• Delete a school year. |
| 13 | Curriculum & Syllabus Management | Program Management Screen | This screen allows the admin to view and manage curriculum programs.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Programs, Vietnamese programs, English programs, and Inactive programs.<br/>• A grid layout of program cards displaying target age group, language, program name, description, lessons count, and status badge.<br/>• A filter bar to filter programs by language, target age group, and status.<br/><br/>**Available Actions:**<br/>• Filter programs using the filter bar.<br/>• Search programs by name.<br/>• Create a new program.<br/>• Update a program.<br/>• View lessons list under a program.<br/>• Delete a program. |
| 14 | Lesson Content Management | Lesson Management Screen | This screen allows the admin to view and manage lessons within curriculum programs.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Lessons, Active lessons, and counts for categories like Oral Motor or Vocabulary.<br/>• A table displaying lessons with Lesson ID (LSN-xxx), Lesson Order, Lesson Name, Program Name, Target Skill (Pronunciation/Oral Motor/Vocabulary/Communication), Estimated Duration (minutes), and Status.<br/>• A filter bar to filter lessons by program, target skill, and status.<br/><br/>**Available Actions:**<br/>• Filter lessons using the filter bar.<br/>• Search lessons by name.<br/>• Create a new lesson.<br/>• Update a lesson.<br/>• View exercises list under a lesson.<br/>• Delete a lesson. |
| 15 | Exercise & Question Bank | Exercise Management Screen | This screen allows the admin to manage exercises and their corresponding question lists in a master-detail layout.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Exercises, Total Questions, Active Exercises, and Active Exercise Types.<br/>• A master table displaying the list of exercises with ID, name, lesson, type, skill, difficulty (Easy/Medium/Hard), language, duration limit (seconds), status, and question count.<br/>• A detail table displaying the list of questions for the selected exercise, showing question ID, question sentence, answer sentence, input type (Speech, Multiple Choice, Repeat, Oral Motor, etc.), instructions, and audio/image status.<br/>• A filter bar to filter exercises by lesson, type, difficulty, language, and status.<br/><br/>**Available Actions:**<br/>• Filter exercises using the filter bar and search by name.<br/>• Create/Update/Delete exercises.<br/>• Manage questions under the selected exercise (create a new question, edit question details, attach audio/image, and delete a question). |
| 16 | Exercise Type Management | Exercise Type Management Screen | This screen allows the admin to view and manage exercise types/formats supported in VR.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Exercise Types, Active types, and Inactive types.<br/>• A grid list displaying exercise types with custom icon, Type ID, Type Name, Description, Status, and Created/Updated Date.<br/>• A filter bar to filter exercise types by status.<br/><br/>**Available Actions:**<br/>• Filter exercise types using the filter bar.<br/>• Search exercise types by name.<br/>• Create a new exercise type.<br/>• Update an exercise type.<br/>• Delete an exercise type. |
| 17 | VR Learning Result Monitoring | Learning Result Management Screen | This screen allows the admin or teacher to monitor children's practice attempts and provide clinical feedback.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Sessions, Average Accuracy (PCC/MLU), High Score count, and Progress Trend.<br/>• A table displaying learning results with Result ID, Child Name, Lesson/Exercise Name, Score (%), Attempt Number, Duration (seconds), Completed Date, and Status (Completed/NeedReview/Failed/InProgress).<br/>• A filter bar to filter learning results by status, difficulty, target skill, date range, and child name.<br/><br/>**Available Actions:**<br/>• Filter learning results using the filter bar.<br/>• Search learning results by child or exercise name.<br/>• View learning result details (ASR speech transcript comparison, timeline event logs).<br/>• Play recorded child voice audio and simulated VR replay data.<br/>• Write/Edit clinical feedback received by parents. |
| 18 | Phoneme Analysis Detail | Pronunciation Detail Screen | This screen provides detailed phonetic transcripts and error analysis for child practice attempts.<br/><br/>**Displayed Information:**<br/>• Summary cards displaying Total Analyzed Phonemes, Average Accuracy, Common Issue phonemes, and classified issue types.<br/>• A table displaying phonetic records with Detail ID, Child Name, VR Attempt ID, Expected Phoneme, Actual Phoneme, Accuracy Score, and Issue Type (Lỗi nhận diện).<br/>• A filter bar to filter phonetic details by issue type, accuracy range, child, and exercise.<br/><br/>**Available Actions:**<br/>• Filter pronunciation details using the filter bar.<br/>• Search pronunciation details by phoneme.<br/>• View error detail explanation modals (showing mouth/tongue placement illustrations and audio play). |
| 19 | Therapy Progress Analytics | Progress Analysis Screen | This screen provides visual graphs tracking a child's speech therapy progress over time.<br/><br/>**Displayed Information:**<br/>• A child selector dropdown filter.<br/>• Charts (when a child is selected) showing Pronunciation Accuracy Trend (Line Chart) and Practice Duration & Completed Exercises (Bar Chart), alongside strengths, weaknesses, and recommendations.<br/>• A table (when no child is selected) showing child names, average scores, total practice time, completed exercises, and progress status (Improving, Stable, Need Support).<br/><br/>**Available Actions:**<br/>• Select child.<br/>• Filter list by progress status.<br/>• View detail progress modal (strengths, weaknesses, recommendations).<br/>• Export progress data to Excel spreadsheet. |
| 20 | Academic Report Generator | Report Generator Screen | This screen allows the admin or teacher to generate and export formal progress reports.<br/><br/>**Displayed Information:**<br/>• A filter section with child selector and report time period dropdowns.<br/>• A report preview card showing teacher name, progress chart (Area Chart of scores), and a list of recent session items.<br/><br/>**Available Actions:**<br/>• Apply filters to generate previews.<br/>• Click the Export button to export and download a professional PDF report. |
| 21 | Admin Profile Settings | Account Settings Screen | This screen allows the admin to manage their profile and credentials.<br/><br/>**Displayed Information:**<br/>• A profile form with avatar, email (read-only), full name, phone number, gender, and role label (Admin).<br/>• A change password form.<br/><br/>**Available Actions:**<br/>• Edit and save profile information.<br/>• Update login password. |
